# 全课重点

## 一句话理解这门课

本课程以职业道德为基础，以文创产品设计与运营为主线，系统讲授从创意构思、市场调研、设计实现，到品牌建设、知识产权保护的完整文创产业链知识。

## 重点概念速查

| 概念 | 含义 | 相关章节 |
|------|------|---------|
| 职业道德 | 从业者在职业活动中应遵循的行为规范，以自律为主，通过公约守则形式体现 | 第1章 |
| 集体主义原则 | 职业活动中将集体利益置于首位，个人利益与集体冲突时学会权衡 | 第1章 |

| 道德风险 | 信息不对称导致一方行为损害另一方利益的经济学现象 | 第3章 |

| 文创产品 | 依托文化资源，通过创意转化和科技提升形成的以文化为核心内容的产品，兼具商品、文化、创意三重属性 | 第5章、第29章 |
| 设计思维 | 以用户为中心的问题解决方法，包含同理心、定义问题、创意发散、原型制作四步骤 | 第5章、第18章 |

| 生命周期评估（LCA） | 从"摇篮到坟墓"全面评估产品全生命周期环境影响的系统工具 | 第12章 |
| 3R原则 | 可持续设计的基本原则：减量（Reduce）、再利用（Reuse）、再循环（Recycle） | 第12章 |

| 增强现实（AR） | 在现实世界中叠加虚拟图像或信息，用户仍可看到真实环境 | 第16章 |

| 概念设计 | 将模糊概念转化为具体设计蓝图的思维过程，是设计的起点与核心 | 第18章 |

| 合理使用 | 版权法允许在教育、评论、新闻报道等特定场景下无需授权使用作品 | 第30章 |

| UGC（用户生成内容） | 用户自发创作并分享的内容，可增强产品可信度与口碑传播 | 第8章、第28章 |
| AB测试 | 比较两种方案实际效果的实验方法，用于选出更符合用户需求的设计 | 第28章 |

## 方法论清单

- **培养职业道德三步法**：认识重要性并内化为准则 → 通过学习实践提升修养 → 以身作则影响周围环境
- **职业道德风险管理四环节**：识别 → 评估 → 控制 → 监督
- **设计思维四步骤**：同理心（理解用户） → 定义问题（明确目标与场景） → 创意发散（头脑风暴，不过早设限） → 原型制作（具象化并测试）
- **创意激发四种方法**：头脑风暴（无错原则）、思维导图（放射性扩展）、逆向思维（从反向角度创新）、角色扮演（模拟目标用户）
- **文创产品定制设计五步流程**：文化选取与剖析 → 产品调研 → 形成初稿预案 → 审查批判改稿 → 建模出效果图及工艺选取
- **市场调研实操四步骤**：制定计划 → 收集数据 → 分析解读（描述性→推断性）→ 撰写报告
- **个人职业道德素养提升四步法**：自我反思 → 持续学习 → 实践应用 → 以身作则
- **品牌定位三条路径**：挖掘独特性 → 提炼核心价值 → 制定个性化定位策略
- **色彩搭配黄金法则**：主色60% + 辅助色30% + 点缀色10%
- **市场定位四步骤**：目标市场选择与细分 → 消费者需求分析 → 竞品分析 → 品牌真言创建
- **知识产权保护计划四步**：全面市场调研了解侵权风险 → 建立知识产权清单 → 及时获得法律保护 → 建立监控与应对机制
- **知识产权纠纷解决优先级**：协商和解 → 行政裁决 → 司法诉讼 → 仲裁解决
- **打样制作评估流程**：制定评估计划 → 定性分析（专家评审+用户体验测试）→ 定量分析（性能测试+市场调研）→ 综合分析 → 制定改进策略

## 重点案例

- **故宫文创**：深挖传统文化元素与现代设计结合的标杆，涵盖产品策略（节日主题、定制礼品）、推广策略（社交媒体+线下展览）、AR技术应用，体现文化性与商业性的成功平衡
- **南京冠生源事件**：职业道德缺失的经典反面案例，导致企业破产、个人失信、社会信任体系受损
- **苹果公司**：从设计理念（简洁易用）到品牌核心价值（创新、简洁、用户体验）、营销策略（差异化定位）的综合案例，以及苹果 vs 三星知识产权诉讼
- **《哪吒之魔童降世》《流浪地球》**：中国文创产品提升国家文化软实力的代表，体现文
