# 第 2 章：：文创产品设计思维与技能提升
## 1. 本期主题与核心观点

本期课程围绕**文创产品设计创新的八大路径**展开，旨在帮助文创设计师与运营者建立系统的设计思维框架。核心观点如下：

- 文创设计既需要情怀与坚持，也必须兼顾商业与发展，实现多方共赢
- 所有文创设计的底层逻辑都基于**用户需求**，从需求出发，经过开发，到消费，最终形成文化认同
- 新消费趋势的本质是挖掘潜在需求、响应用户对美好生活的追求，消费升级（而非表面的消费降级）是长期趋势
- 八大设计路径（色调、纹饰、形态、功能、材质、包装、情感、主题）均可作为独立突破口，且可两两或多重组合使用
- 文创开发必须"做中学"，面向实际，不能仅停留于书本与策划案

## 2. 关键要点

### 路径一：色调
- 色彩是最直接的感官体验，对用户感知产品价值有高效影响
- 高饱和度、色相明确的颜色（红、黄）传递廉价感与快节奏感（快餐品牌典型用色）
- **降低饱和度**或使色相模糊，可削弱颜色对情绪的冲击，使产品显得更高档
- 奢侈品牌偏用黑白/灰阶广告，本质是降低色彩的情绪影响，提升高级感
- 品牌色使用策略：在自有渠道/发布会用黑白彰显高端；在竞争嘈杂的公共场所用彩色提升辨识度
- 文创产品的莫兰迪色调（高级灰）应用已成趋势，可借鉴《延禧攻略》等案例

### 路径二：纹饰
- 平面化纹饰制造成本低，是早期文创设计的主流方式
- 中国传统文样"有图必有意，有意必吉祥"，纹饰在文博文创中尤为重要
- 纹饰使用需有文化依据，设计师须研究纹饰的历史起源与使用时期（如钱文盛于三国两晋，不可随意用于其他朝代产品）
- **三种平面化纹饰应用方式：**
  1. **贴图式**：直接将纹饰贴附于产品载体，适合周边商品、礼品，成本低但创意有限
  2. **插图式**：基于地域文化元素进行原创插画创作，系列化应用
  3. **立体转平面式**：将立体文物形态提炼为平面纹饰，结合新工艺完成转化
- 纹饰从不过时，运用得当是高效的设计语言；描线能力与纹饰提炼是文博文创设计师的基本功

### 路径三：形态
- 形态创新多属于工业设计类文创，开发成本较高但原创度更高
- 形态不能孤立看待，必须结合材质与色调综合考量
- 形态设计应兼顾**美观与实用**，以功能需求为出发点
- 反例：体量过大、脱离消费场景的产品（如过大的木雕拐杖）注定是小众工艺品，而非大众文创
- 盲盒类产品为形态设计的典型代表，开发需至少4+1款（含隐藏款），模具成本高，需评估投入产出比
- 增加角色**故事性（叙事性）**是提升盲盒产品吸引力的关键

### 路径四：功能
- 功能创新常被设计师忽略，但功能突破本身就是创意点
- 沙利文经典论断"形式追随功能"至今不过时
- **四种趣味功能打造方式：**
  1. **置换**：将A产品外观与B产品功能互换，产生趣味感（如镜头造型水杯）
  2. **动态**：使用过程中产生形态变化（如随酱油减少显现富士山图案的调料碟、越擦越出造型的橡皮擦）
  3. **转换**：改变产品的使用角度或方式（如旋转90度的蚊香支架）
  4. **叠加**：A产品功能+B产品功能相加（如发光冰箱贴）
- 注意叠加功能类产品往往本质是小家电而非文创，需控制定位边界，评估投入产出比

### 路径五：材质
- 材质包含材料、质地与制造工艺，可引入工业设计CMF体系（Color色彩、Material材料、Finishing表面处理）
- **四种材质创新方式：**
  1. **选择材料**：选用契合文物气质的替代材料（如用树脂模拟芙蓉石）
  2. **置换材料**：将A领域常见材料换至B领域，置换需有合理逻辑（如将紫砂壶材质用于咖啡器具——因同样需要吸附香气）
  3. **组合材料**：A材料+B材料叠加（如不锈钢露营杯+软木塞保护套）
  4. **改变质地**：利用视觉与触觉的通感反差制造惊喜（如外形似鹅卵石实为柔软沙发）
- 非遗/手工艺材质（第五类）：结合非遗工艺进行适当创新，而非原封不动地使用（如将造纸非遗工艺加入咖啡渣，开发咖啡店周边包具）

### 路径六：包装
- 好的包装设计本身就是文创产品；64%的消费者会因包装改变而尝试购买新产品
- 包装设计的三大价值：文化彰显、品质传递、促进销售
- 包装不等于越繁复越好，需评估成本对定价策略和受众的影响
- **旅游文创包装的四个设计方向：**
  1. **体现文化特性**：运用地域刺绣、竹编等工艺，彰显在地文化（如竹叶青茶叶包装用蜀绣）
  2. **隐喻商品品质**：通过包装语言传递产品属性（如药妆包装强调功效感）
  3. **传递新鲜感**：以新颖形式制造视觉惊喜（如剪纸风格+科技感的月饼包装）
  4. **融合使用功能**：便携、即时、即用、仪式感（如T恤盒折叠成衣架；藏红花包装配镊子和底座，营造仪式感）

### 路径七：情感
- 情感设计目标是探寻产品与用户之间的意义与关系，超越"好不好看"的层面
- **六种情感设计维度：**
  1. **情绪共鸣**：捕捉当代用户心态（如"休息是合法的"打工人徽章）
  2. **沉浸体验**：营造环境氛围感（香氛类产品、沉浸式空间设计）
  3. **心流状态**：引导用户进入专注内心的体验（寺庙文创、抽签/刮刮卡设计）
  4. **回忆触发**：复古风格唤起集体记忆（卡带磁带造型文创）
  5. **下意识冲动**：捕捉童年本能行为（起雾玻璃UI界面设计）
  6. **反思引导**：促发社会责任感（反浪费餐盘壁画设计）

### 路径八：主题
- 主题式设计类似命题作文，围绕某一具体文化资源、产品形象或材料限定展开
- 主题可以是一个特定IP形象（如黑玉子/黑鸡蛋）、一种材料（如粘土）或一段文化内容
- 主题确立后，可向外延伸联名、周边、场景化开发，形成系列产品矩阵

## 4. 实操建议

- **建立纹饰研究习惯**：做文博文创前，系统研究纹饰的历史起源、盛行时期与文物依据，确保设计有文化立足点，能回答"为什么用这个纹饰"
- **掌握矢量转化技能**：使用CorelDRAW等工具的"轮廓描摹-线条图"功能，可将实拍照片中的书法/图案快速转化为矢量文件，作为纹饰设计的基础素材（演示了白鹤梁题字的转化流程）
- **色调应用原则**：产品定位高端时，降低饱和度、模糊色相；需要大众传播或强调性价比时，使用鲜明品牌色；减少颜色种类本身就是提升设计质感的有效手段
- **功能突破优先于形式**：在设计立项时，先问"这个产品的功能有没有可以置换、动态化、转换角度、叠加的空间"，功能创新往往比纯造型创新更具竞争力和溢价空间
- **盲盒开发成本控制**：拆件越多成本越高，模具费用是主要成本项，立项前须严格估算销量预期与开发投入的正相关性，避免亏损
- **包装设计不要与产品割裂**：包装应纳入文创产品整体开发流程，结合地域文化工艺（刺绣、竹编等），使包装本身也具有留存和二次使用价值（如腰封变书签）
- **置换材料需有逻辑支点**：材料置换不是为了猎奇，必须找到两种材料或两个场景之间的内在共性（功能匹配、文化关联、使用场景相近），否则会显得牵强
- **情感设计要洞察真实需求**：从用户的情绪、记忆、下意识行为中捕捉设计切入点，高阶情感设计（如反思类）可触达更深层的文化共鸣
- **做中学，不迷信策划案**：文创经验具有强烈的地域性，必须通过实际落地项目积累经验，持续与真实消费者互动，动态调整方向

# 文创产品设计思维与技能提升

## 核心理念
### 设计原则
- 情怀与商业兼顾，实现多方共赢
- 底层逻辑：用户需求 → 开发 → 消费 → 文化认同
- 新消费趋势本质：挖掘潜在需求，响应对美好生活的追求

- 八大路径可独立突破，也可两两或多重组合使用
- 必须"做中学"，面向实际落地

## 路径一：色调
### 色彩感知原理
- 高饱和度（红、黄）：传递廉价感与快节奏感
- 降低饱和度 / 模糊色相：削弱情绪冲击，提升高档感

- 竞争嘈杂的公共场所：彩色提升辨识度
- 减少颜色种类本身即可提升设计质感
### 应用趋势
- 莫兰迪色调（高级灰）已成文创主流
- 参考案例：《延禧攻略》色调体系

## 路径二：纹饰
### 基本原则
- "有图必有意，有意必吉祥"

### 设计师基本功
- 描线能力与纹饰提炼
- 矢量转化技能：CorelDRAW"轮廓描摹-线条图"功能
- 建立纹饰研究习惯，确保设计有文化立足点

## 路径三：形态

- 模具成本高，需严格评估投入产出比

### 反例提示
