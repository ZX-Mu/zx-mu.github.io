# 第 20 章：文创产品定位、运营、营销
## 1. 本期主题与核心观点

本期围绕文创产品的三大价值类型展开，核心观点是：**文创的灵魂是IP**，所有文创产品开发都必须将文化价值、情绪价值与互联网技术三者融合，才能确保产品具备持续的生命力和市场竞争力。

**最核心公式：文化价值 + 情绪价值 = 产品永生**

## 2. 关键要点

### 文创产品的四大价值维度（本期讲前三条）

**① 文化价值**
- 文创产品的文化背书必须有足够的历史沉淀，通常需要1000年以上才能称为文化
- 单纯的文化价值不足以支撑销售，因为知道的人少，认知度低
- IP是文化的载体，一个景区/产品应优先绑定大众熟知的IP（如秦始皇），而非冷僻的地方IP
- IP的造像没有固定形式，不要固化，避免形成思维定势
- 文创策划原则：**去宗教化，强民俗化**

**② 情绪价值**
- 情绪价值是推动消费者购买的直接驱动力，人购买行为本质上受情绪感染
- 纯情绪产品生命周期极短，更迭速度极快（如天水麻辣烫公仔、蒜鸟公仔）
- 情绪价值的激发方式有两种：**场景激发**（到了特定地点自然产生）、**文化激发**（由深厚文化底蕴自然唤起）
- 情绪产品开发要求：定位精准、快速推出、控制库存

**③ 文物复刻**
- 将文物按比例（1:1或缩小版）还原制作成文创产品
- 最常见品类：青铜器复刻
- 核心弱点：离开原场景后认知度迅速消失
- 解决方案：必须结合互联网技术（二维码/三维码/NFC），让技术承载文物背后的历史故事

**④ 产业价值（本期未展开，下期内容）**
- 核心思路：土特产/特色产品的文创化
- 讲师自称"产业文创"概念创始人

### 文创产品开发的核心逻辑

- 所有类型的文创产品，最终都必须与情绪价值融合，因为文创是卖给人的
- 好的文创产品必须有主题、有说明，不能只靠陈列
- 快消品（水、酒、奶茶等）的包装方向，核心就是文化创意
- 情绪产品 + 文化价值 = 可延续传播的长寿产品

### 营销方法论

- **疑问即是卖点**：触发消费者疑问，就激活了购买欲望
- 一看一问一交流，购买前提即成立
- 产品命名、陈列都应主动制造疑问点，而非被动等待消费者理解

## 4. 实操建议

- **选IP要选大众熟知的**：与其只做小众地方IP，不如绑定秦始皇、女娲、故宫等全民认知IP，再关联地方文化
- **产品开发前先判断价值类型**：明确该产品主打文化价值还是情绪价值，并有意识地将二者融合
- **情绪产品要快、准、少量**：快速上市、精准定位目标用户、控制库存，避免积压
- **所有文创产品必须结合技术**：二维码、三维码、NFC等技术可以承载产品背后的故事，激活用户记忆，跨越场景限制
- **文物复刻必须配故事**：复刻品本身没有溢价，通过扫码讲述文物历史才能构建价值感
- **产品必须有主题和说明**：好的文创产品不能只靠陈列，要主动讲故事、营造氛围
- **主动制造疑问**：产品名称、卖场陈列要能触发"这是什么？"的好奇心，疑问本身就是最强卖点
- **避免两个误区**：① 不要只做冰箱贴等传统地摊文创，赛道已饱和；② 文化类产品要去宗教化、强调民俗化，否则难以商业落地
- **产业文创方向**：将地方土特产进行文创化改造，是目前最大的蓝海赛道之一，值得重点布局
