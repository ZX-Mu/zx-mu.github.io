# 第 22 章：文创产品定位、运营、营销
## 1. 本期主题与核心观点

本期为课程答疑与案例展示环节，核心围绕"全域流量生态构建"与"文创产品如何与文化IP深度结合"展开，同时解答了多类行业（矿业、动物园、高校、农产品、中医等）的文创开发思路。

**核心论断：**

- **文创必须生活化，生活化的最好体现方式就是产业化。** 产业化才能使文创成为文旅文化传播的重要载体。
- 全域流量的本质：文创产品不能只停留在景区内部，要渗透到日常生活的方方面面。
- **文创 = 文化底蕴基础上的创造**，不是简单的插画+模具，是文化的仪轨与载体。
- 情绪价值（潮玩）生命周期短，必须与文化深度结合才能持久。
- 场景是变现的前提：文创产品必须在对应的文化场景中销售，脱离场景则难以变现。

## 2. 关键要点

- **全域流量生态构建**：文创要从景区走向生活，实现产业化落地，形成"产业基地景区化、产品旅游商品化、游学产品化"的沉浸式消费场景。

- **文创 vs 潮玩的本质区别**：
  - 潮玩：强调情绪价值，文化底蕴少，生命周期短（以哪吒、悟空IP为例，热度过后迅速衰退）。
  - 文创：以文化底蕴为根基，情绪价值与文化相结合，才具备持久生命力。

- **文创产品设计原则**：
  - 必须讲清楚文化脉络，避免被误解为迷信或低俗。
  - 产品要有AR/扫码互动功能——"没有码的文创没有灵魂"。
  - 坚持东方美学风格，避免盲目模仿日式漫画或丑陋化造型（如Labubu风格）。
  - 去宗教化、强民俗化：佛像等元素可Q版化、卡通化，但不得用于宗教宣传。

- **IP开发核心原则**：
  - IP是最核心的差异化竞争手段，文化IP的差异才是农产品、同类景区之间真正的竞争壁垒。
  - 传统文化IP属于全民共有（女娲、伏羲等），任何人均可创作，但不能直接复制他人的创意设计作品（著作权保护的是具体表达，不是文化素材本身）。
  - 产品设计要具备唯一性与仪式感，成为文化仪轨的载体。

- **版权保护要点**：
  - 文化素材本身无版权（泰山、少林寺、伏羲庙等古迹）。
  - 对素材进行原创设计后，该设计作品受著作权（美术作品著作权）保护。
  - 地图类产品：普通地图需经测绘局审批；手绘创意地图无此限制。

- **变现的前提条件**：
  - 文创产品必须在对应文化场景中销售（如故宫文创在故宫卖，景区文创在景区卖）。
  - 脱离场景，同款产品价格可从298元跌至29.8元，且无人问津。

- **策划运营思维的核心**：
  - 设计只是工具，核心是：朝哪个方向设计、如何销售、如何产生复购、产品间能否形成关联与闭环。
  - 文创人员必备素质：熟悉中国传统文化 + 情怀（对祖国山河文化的热爱）。

- **学习路径建议**：
  - 不要试图掌握所有中国文化，应聚焦于所在城市或目标领域，深耕一个方向。
  - 文化传承需要有"师父"，不仅仅是课堂学习。

## 3. 重要案例或数据

**案例一：甘肃天水——黄河文创系列**
- 天水是黄河文明起点（大地湾遗址，距今约8700年，全球已知最早文明遗址之一）。
- 讲师在天水高铁站一楼、二楼运营3家文创店（三皇文创店），为全国高铁站贵宾室中唯一设有文创的单位，受天水市文旅局委托开发运营。
- 代表产品：
  - **伏羲决策系统（伏羲64卦）**：以挂台山8个卦台为原型，融合挂台山的土与伏羲庙香灰制作，文化来源为《河图洛书》，定位为"决策工具"而非算命。
  - 冰箱贴、手机架、文化礼包等，均搭载AR扫码动画，销售火爆。
  - 产品已获奖，郑海霞等名人曾在店内购买。
- 合作项目：为"环西部列车游"（甘肃、宁夏、敦煌莫高窟沿线）开发车载文创产品，合同在谈中。

**案例二：山西运城——关公文化文创**
- 运城拥有102处国家级重点文物保护单位，被称为"天下国宝第一市"。
- 在运城机场、关帝庙、永乐宫设有文创店（河东文创）。
- 镇店之宝：**《过五关斩六将》大幅原创作品**，扫描作品中的马匹可触发AR动画，呈现关公千里走单骑全程。
- 另有后土祠文创系列（后土祠与"30年河东30年河西"典故相关联）。

**案例三：山东牛蒡茶——孔子IP赋能农产品**
- 产品本身差异化难度大，通过创作故事赋予文化价值：
  - 故事设定：公元前538年，孔子赴老子处问道，马车上装有牛蒡喂马，马吃后力气大增。
  - 产品命名为"问道茶"，配套AI动画短片，扫码可观看。
- 逻辑：同类农产品的竞争壁垒在于文化IP与故事，而非产品本身。

**案例四：高校文创——甘肃工业职业技术学院**
- 讲师与甘肃工业职业技术学院成立"三黄文创研究院"，讲师担任客座教授及商学院院长。
- 产品以"三黄文创研究院出品"标注，借助高校背书提升品牌信任度。
- 结论：高校文创的价值在于借助高校背书，承担城市IP挖掘与文化研究的功能，而非仅出售校名周边。

**案例五：北京故宫IP授权参考**
- 故宫IP联合开发授权费用数百万元起。
- 同款产品：贴有"故宫出品"标签的产品在景区内售价298元，景区门口仿制品售价29.8元。

## 4. 实操建议

**通用文创开发流程：**
- 先确定所在城市/区域的核心文化IP，深度研究其历史文化脉络。
- 围绕IP创作故事，故事要有历史依据或文化逻辑，避免被认定为迷信。
- 产品形态：实物产品 + AR/扫码互动内容，两者缺一。
- 申请著作权（美术作品著作权）保护原创设计，而非专利。
- 必须在对应文化场景中销售，或构建能承载该文化的消费场景。

**各行业文创开发方向：**

| 行业 | 建议方向 |
|------|----------|
| 矿业公司 | ①矿石科普产品；②矿石造型文创；③矿山机械造型文创（参考首钢）；④矿产品形态食品（如煤形糕点） |
| 动物园 | ①明星动物Q版IP衍生品（公仔、服装、挂件）；②动物IP主题食品（犀牛冰激凌等）；重点做IP授权 |
| 高校文创 | 非名校不宜单做校名产品；应与所在城市IP联名，高校承担城市IP挖掘研究功能 |
| 中医类 | 以"食药同源"为核心IP，深入挖掘《黄帝内经》等经典，与当地医药文化场景结合 |
| 农产品 | 不做产品本身差异化，做文化IP差异化：用当地历史人物/传说为产品创作专属故事 |
| 枸杞/养生产品 | 结合产地IP（宁夏、天水等）创作养生文化故事，与地方神话传说关联 |
| 城市文化传播 | 不必在外地复制景点，可以沉浸式文化场景业态（如陕西面馆+陕西文化场景）为载体 |
| 全国景区展示 | 使用手绘创意地图形式，规避测绘审批要求；地图创意设计本身受著作权保护 |

**文创人才核心能力要求（优先级排序）：**
1. 营销思维（变现路径、复购机制、产品闭环）
2. 中国传统文化知识储备
3. 对文化传播的情怀与热爱
4. 设计执行（可外包，但需要掌握方向判断能力）
