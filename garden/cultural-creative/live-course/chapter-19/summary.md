# 第 19 章：文创产品定位、运营、营销
## 1. 本期主题与核心观点

本期课程由具有营销策划、互联网及传统文化研究背景的牛老师主讲，核心主题为**如何让文创产品真正"销"出去**，强调在"销"之前必须先"赢"。

**核心观点：**
- 文创不等于冰箱贴。冰箱贴思维做文创，成功概率极低。当一个产品开始在地摊上售卖，意味着该赛道已进入红海，应立即寻找新方向。
- 文创产品的本质是**文化符号**，不是单纯的"型"（外形复刻），必须有"神"（文化内涵）。脱离场景、缺乏文化支撑的产品只能靠低价竞争，没有出路。
- **赢 = 创造（与众不同）；销 = 把产品卖出去。**有"赢"的能力，才有"销"的基础。
- 做文创策划师，必须具备文化底蕴，对中华历史文化有热爱和深入理解，而不仅仅是平面设计或动画技能。

## 2. 关键要点

**课程结构（三章）：**
- 第一章：文创产品 IP 的定位与价值挖掘
- 第二章：全域流量生态构建与用户运营
- 第三章：实战案例解析

**文创 IP 的四个定位/价值维度：**

1. **文化价值**
   - 文创的核心目的：传播文化记忆点，让人通过产品联想到一段历史或故事。
   - 文创产品背后必须承载一个具体的中国文化故事。
   - 做法：挖掘景区/文物背后的历史叙事，将其转化为产品的灵魂，而非仅复刻外形。
   - 呈现方式须结合互联网技术工具（见下方技术工具部分）。

2. **情绪价值**（本期课程提及，未展开）

3. **文物复刻**（本期课程提及，关联国家文物局"文物活化工程"政策，未展开）

4. **产业价值**（本期课程提及为最重要一项，未展开，将在后续讲解）

**文创产品的三重价值（对景区/机构而言）：**
1. 直接增加景区利润
2. 通过产品将景区文化带出去，形成传播
3. 反向引流——消费者因想购买文创产品，产生前往景区旅游的意愿

**技术工具的三种应用（文化内容的数字化呈现）：**
- 二维码（扫码触发内容）
- 三维码（AR 动画呈现）
- NFC 芯片（手机触碰即触发故事内容）
- 当前 AI 视频工具的出现大幅降低了动画内容的制作成本，为文创产品的数字内容层提供了低成本解决方案。

## 4. 实操建议

- **入场前先判断赛道**：一旦发现目标产品已出现在地摊，立即撤退，寻找新赛道，不要在红海中跟风入场。

- **建立产品护城河**：文创策划的核心竞争力在于文化理解深度，而非设计技术。深入研究目标景区/文物的历史背景，挖掘尚未被开发的文化叙事角度（如永乐宫的吕洞宾养生文化）。

- **产品开发流程参考**：
  1. 实地调研，亲自拍摄文物细节素材
  2. 深挖该文物/场所背后的历史故事与文化意义
  3. 将文化故事转化为产品的核心概念（而非仅复刻外形）
  4. 结合功能性设计，提升产品实用价值
  5. 嵌入技术层（二维码/NFC/AR），将文化故事数字化呈现

- **善用 AI 工具降低内容制作成本**：使用 AI 视频生成工具制作文化故事动画，替代高成本的传统动画制作，降低文创产品数字内容层的开发门槛。

- **明确文创产品的三重使命**：策划时须同时设计好产品如何实现"增加景区利润、传播文化、反向引流"三个目标，单一目标的文创产品策划是不完整的。

- **持续积累文化底蕴**：关注国家文物局"文物活化工程"等政策文件，了解中华人文始祖相关文化体系（伏羲、女娲、炎帝、黄帝等），这是文创策划师区别于普通设计师的核心壁垒。
