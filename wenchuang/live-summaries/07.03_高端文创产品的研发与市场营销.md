# 文创策划运营直播课程 - 第七期学习笔记

## 1. 本期主题与核心观点

本期聚焦高端文创产品的研发与市场营销实战，通过"冬奥五环珐琅尊"案例深度解析产品设计的底层逻辑。核心观点：高端文创产品的成功不在于设计师的个人创意，而在于精准提炼传播概念、把握市场需求场景（空间美学陈设、高端礼赠），以及将复杂信息层次化并转化为产品语言的能力。

## 2. 关键要点

- **硬需求场景定位**：高端文创产品存在两大刚需市场——重要场所的空间美学陈设和高端礼赠市场，单品可实现数千至上万套销量
- **设计师角色重定义**：设计师应"一手牵市场、一手牵技术"，而非依赖个人小创意，核心是连接市场需求与生产供应链
- **信息层次化原则**：将主题相关信息按重要性从核心到边缘排列（如冬奥：五环→北京冬奥→会徽吉祥物→双城元素），确保传播精准高效
- **传播效率公式**：提炼的核心信息越少，传播效率越高，成本越低；最终销量=触达率×产品过关率（成交率）
- **命名即传播**：产品名称需包含全部核心信息（冬奥+五环+珐琅+尊），让人未见产品先知概念
- **资源自动聚集效应**：当产品方向选对、做到足够高度时，媒体、政府等资源会主动跟随（如冬奥主题自然吸引电视台选题）
- **设计执行逻辑**：方向确定后，设计任务就是用产品把核心概念"表现得淋漓尽致"，而非另起炉灶创新
- **传播触达率案例**：去年"燃烧的向日葵"项目实现近10亿次传播触达，证明精准提炼概念的威力

## 3. 重要案例或数据

**冬奥五环珐琅尊案例**：
- **产品定价**：39,800元/件
- **限量发行**：2,022套
- **实际销量**：1,700套左右
- **总销售额**：约6,766万元
- **传播成果**：北京电视台冬奥纪实频道跟踪拍摄一个月专题片
- **设计团队**：钟连盛（从业40+年景泰蓝大师，曾参与2008北京奥运特许商品设计）领衔

**设计元素拆解**：
- 核心视觉：奥运五环颜色（蓝黄黑绿红）正投影叠落，覆盖传统冷冬纹
- 第二视觉中心：纯金打造冰墩墩图案+吉祥物图案
- 细节层：盖环纯银雕刻北京与张家口标志建筑及冬奥运动项目
- 文化融合：冰雪+琉璃工艺、虎年虎足鼎概念、三生琉璃底座

## 4. 实操建议

1. **启动项目前先做信息层次梳理**：列出主题的所有关键词（如北京、2022、冬季、奥运会），按重要性从核心到边缘排序，找出最高频词汇和最核心视觉元素

2. **用"核心概念提炼法"命名产品**：将最重要的3-7个关键词组合成产品名，确保名称本身就是完整的传播信息（参考：冬奥+五环+珐琅+尊）

3. **锁定硬需求场景再定价**：针对空间美学陈设和高端礼赠市场，可大胆定高价（3-5万元级别），配合限量发行策略

4. **设计前明确"传播效率优先"**：每增加一个传播信息点都会增加成本，宁可简化到极致也不要信息冗余，用一句话/一个核心概念打透市场

5. **主动对接资源而非等待**：当产品主题契合社会热点（如奥运、国庆等盛事）时，主动联系媒体提供选题，资源会因内容价值而聚集

6. **建立"双指标"评估体系**：项目推进中持续监测传播触达率和产品成交率两个核心指标，前者靠精准概念，后者靠产品品质
