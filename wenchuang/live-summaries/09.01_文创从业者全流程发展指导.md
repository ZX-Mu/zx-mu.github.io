# 文创从业者全流程发展指导（一）- 学习笔记

## 1. 本期主题与核心观点

本期课程由一位从事文创产品设计多年的创业者分享，重点讲解文创设计的前两个核心步骤：**了解认知**和**文化溯源**。讲师强调文创设计不是简单的元素堆砌，而是需要对文化有深度热爱和系统研究，通过找准一个文化元素进行多维度延展，可以源源不断地产出设计作品。讲师以自己对宋代瓷器文化的深入研究为例，展示了如何从单一元素延展出灯具、调料瓶、花器、首饰等多个产品系列。

## 2. 关键要点

- 文创设计师职业认定是近年才建立的体系，行业正处于规范化发展阶段
- 完整的文创闭环包括：设计→生产→销售→运营四个环节
- 文创设计第一步是"了解认知"：需要对文化有真正的兴趣、热情和传播责任感
- 深度调研是必修课：每个项目都要研究风土人情、阅读资料、参观博物馆、访谈手艺人
- 文创设计第二步是"文化溯源"：从思想美学、历史渊源、时代背景、工艺特征、寓意象征等多层面挖掘
- 设计延展思维：找准一个元素后，先放开想象其在日常生活中的各种应用场景，再找最佳结合点
- 文化溯源分为抽象（哲学、美学、宗教等形而上）和具象（绘画、雕塑、建筑等形而下）两大类
- 要提取与当代精神价值观相吻合的优秀传统文化，而非全盘照搬

## 3. 重要案例或数据

**案例一：宋代瓷器系列产品延展**
- 基于对宋代瓷器的热爱，系统研究宋代五大名窑（汝、哥、定、钧、官）
- 从经典瓶型（玉壶春瓶、梅瓶、胆头瓶等）延展出多个产品线：
  - "最美宋瓷倩影"系列灯具（吊灯、落地灯、台灯）
  - 玉壶春瓶造型的桌面调料瓶组合
  - 半切瓶型的壁挂花器
  - 瓷器开片纹理的首饰设计
  - 中西结合：宋瓷曲线+蒙德里安直线构成的多宝阁
  - "美人妆"系列：梅瓶造型延展的化妆镜、化妆刷、眼影等

**案例二：汴京四梦系列**
- 灵感来源：开封现状与《东京梦华录》描述的落差引发思考
- 四个主题产品：
  - 幽梦：香薰瓶（唤醒对宋代繁华的想象）
  - 惨梦：镂空灯具（结合官瓷开片纹理，象征历史破碎）
  - 现实：笔具系列（书写新篇章，全手工制作）
  - 吉梦：葫芦瓶造型香炉（祈愿大吉）

**案例三：汾酒集团项目**
- 虽然讲师本人不喝酒，但认真研读厚重资料
- 从酒文化中的诗词文化切入，以文人角度设计酒具产品

**案例四：陕西历史博物馆参观**
- 朋友无麦克风、纯人声激情讲解，吸引全馆参观者围观
- 体现文化传播者的热情和感染力

**讲师背景数据**
- 2016年底开始创业，服务博物馆、景区、企事业单位
- 2019-2020年开设线下直营店
- 目前在北京核心位置拥有多家直营店
- 2013年参加故宫创意设计大赛，"功能箱包"成为成名作品

## 4. 实操建议

**调研阶段**
- 每到一个地方，第一站必去博物馆，深度参观而非走马观花
- 阅读项目相关的所有资料文献，即使对主题本身不感兴趣也要完整研读
- 与当地手艺人深度交流，了解现状、制作工艺、销路市场
- 有条件的情况下亲手体验制作过程
- 全方位考察当地加工工艺和制造工厂资源

**设计思维训练**
- 建立个人文化知识体系：可以做一条从远古到现代的工艺美术史梳理
- 练习延展思维：选定一个文化元素后，先不设限地想象其在日常生活中的所有可能应用
- 寻找最佳结合点：从众多想象中筛选出最合理、最有市场价值的设计方向
- 一个元素可以持续延展出多个产品系列，不要急于换题材

**文化溯源方法**
- 从抽象层面入手：研究哲学思想（如天人合一）、美学理念（如意境美）、宗教文化（儒释道）
- 从具象层面入手：研究绘画、雕塑、书法、建筑、景观、工艺美术等可见形式
- 多维度溯源：思想美学、历史渊源、时代背景、工艺特征、寓意象征
- 筛选标准：提取与当代价值观相吻合的优秀文化内容

**职业发展路径**
- 关注文创策划设计师等新兴职业认定机会
- 构建完整业务闭环：设计→生产→销售→运营
- 积累代表作品，参加行业大赛提升知名度

## 三、文创设计四步骤之二：文化溯源
### 溯源维度
- 思想美学角度
- 历史渊源
- 时代背景
- 工艺特征
- 寓意象征

### 溯源原则
- 提取与当代精神价值观相吻合的文化
- 传承优秀传统文化（习总书记指示）
- 取其精华，去其糟粕
- 传统文化内容多元，需要筛选

### 文化的广义理解
- 人类一切社会现象和内在精神的既有概念
- 涵盖人类从过去到未来的历史
- 基于自然基础上的所有人类活动
- 包含衣食住行等物质表象和精神内在

### 溯源分类
#### 抽象层面（形而上）
- 哲学：天人合一等思想
- 美学：自然美、意境美等
- 宗教：儒家学说、佛道文化

#### 具象层面（形而下）
- 绘画
- 雕塑
- 书法
- 建筑
- 景观
- 工艺美术
- 可在博物馆、展览、书籍中找到详细内容

### 个人知识体系建立
- 可做历史梳理：远古彩陶→汉代画像石/砖/漆器→唐代白瓷/金银器→元明青等
- 系统梳理利于后期产品延展
- 可选择一个分支深入，也可做整体梳理
