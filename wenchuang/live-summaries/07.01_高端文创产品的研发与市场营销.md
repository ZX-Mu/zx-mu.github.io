# 文创策划运营直播课程学习笔记

## 1. 本期主题与核心观点

本期聚焦高端文创产品（以景泰蓝为例）的研发与市场营销。核心观点：文创产品开发应以市场需求为导向，而非工艺导向；需要将现代设计思维与传统手工艺深度融合；建立技术壁垒和供应链壁垒是企业长期发展的关键。

## 2. 关键要点

- 景泰蓝作为明清宫廷技艺，目前全国从业者不足3000人，具有稀缺性和高端定位
- 文创是宽泛概念，不仅限于旅游纪念品，影视、产品设计等都属于文创范畴
- 产品研发必须以消费市场为导向，而非以工艺为导向，这是底层逻辑
- 造型设计壁垒薄弱易被抄袭，真正的壁垒在于底层技术研究（如材料、工艺专利）
- 设计与非遗结合是创新方向，需整合院校设计资源、大师工艺、生产企业和营销体系
- 国礼级产品开发经验：从2017年"一带一路"到2022年冬奥会，展示了高端文创的市场价值
- 成功案例参考：青岛啤酒博物馆将单一品类做到年销售4.6亿元
- 跨界合作是趋势：与工艺美术大师、艺术家、影视IP等顶流资源合作

## 3. 重要案例或数据

**景泰蓝历史地位：**
- 太和殿金銮宝座前陈设器12件全部为景泰蓝（宝象2个、麒麟2个、仙鹤2个、香亭2个、香几4个）
- 在故宫存续约500年（1406-1912年）

**国礼案例：**
- 1952年：林徽因、钱美华团队创作新中国第一份国礼
- 2017年：一带一路"四海同庆"（讲师主导创作）
- 2022年：北京冬奥会国礼30件（讲师团队创作）
- 2023年：一带一路国礼项目（讲师负责推导）

**产业数据：**
- 讲师从事工业设计十几年，文创领域十年，专注景泰蓝八年（2016年至今）
- 景泰蓝工艺108道工序，8大工序，基本100%手工制作
- 青岛啤酒博物馆单品类年销售额4.6亿元

## 4. 实操建议

1. **市场调研优先**：产品开发前必须进行目标用户研究和产品线规划，避免以工艺为导向的盲目开发

2. **建立技术壁垒**：投入底层研究（材料、工艺、专利），参考哪吒电影的专利布局策略

3. **整合资源平台**：构建"院校设计+非遗大师+生产企业+营销体系"的四方合作模式

4. **聚焦细分领域**：学习青岛啤酒博物馆案例，将单一品类做深做透

5. **寻求顶流合作**：主动对接工艺美术大师、艺术家、影视IP等头部资源

6. **国际化思维**：产品设计需考虑国际市场需求和文化共通性

7. **供应链建设**：重视供应链壁垒的构建，这是长期竞争力的保障
